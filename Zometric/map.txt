Zometric CRM mobile app features secure Django token authentication, Picsart-style masonry gallery for files, share integration, and multi-file upload with deduplication. The app uses 3 bottom tabs (Home/Files, View/Gallery, Profile) with RTK Query for all API calls and TypeScript for type safety.â€‹

Backend Endpoints & Auth Flow run : on : " http://127.0.0.1:8000/"
Django backend provides these exact endpoints with Token auth (permanent until explicit logout):

| Endpoint       | Method   | Auth  | Purpose                      |
| -------------- | -------- | ----- | ---------------------------- |
| /auth/signup/  | POST     | None  | Create user + auto token     |
| /auth/login/   | POST     | None  | Login + return token         |
| /auth/logout/  | POST     | Token | Delete token + logout        |
| /auth/profile/ | GET      | Token | User data + tokens list      |
| /files/        | GET/POST | Token | List files / Upload w/ note  |
| /files/        | POST     | Token | File upload (note mandatory) |
